================================================
GNS のどこがいいの？
================================================

年間のサブスクリプションが不要。
対応しているファームウェアであればマルチベンダでネットワークトポロジをシミュレーションすることができる。
例えばかの有名な Juniper の JUNOS を動かすこともできるらしい。
VIRL は Cisco 製品のみの対応だからね。
まあでも Cisco IOS で凝ったことやろうとしたら VIRL のほうがいいと思う。
